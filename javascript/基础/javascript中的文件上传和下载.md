##  javascript 中的文件上传和下载

之前接触的文件上传和下载一直是用到的组件库来实现的，所有没有机会过多的去探究前端在文件的上传和下载操作的基础原理。最近一段时间正好项目中需要自己去实践探究一下，下面就记录一下自己的总结

### 文件操作
首先介绍下在js在文件操作方面的一些基础api.
- **Blob:**
> Blob 对象表示一个不可变、原始数据的类文件对象。它的数据可以按文本或二进制的格式进行读取，也可以转换成 ReadableStream 来用于数据操作。
- **File:**
> 文件接口提供有关文件的信息，并允许网页中的 JavaScript 访问其内容。File 接口基于Blob，继承了 blob 的功能并将其扩展使其支持用户系统上的文件。
- **FileReader:**
> 允许Web应用程序异步读取存储在用户计算机上的文件（或原始数据缓冲区）的内容，使用 File 或 Blob 对象指定要读取的文件或数据。
- **FormData:**
> 接口提供了一种表示表单数据的键值对 key/value 的构造方式,`multipart/form-data`
- **ArrayBuffer:**
> 对象用来表示通用的、固定长度的原始二进制数据缓冲区。
